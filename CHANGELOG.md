# Changelog

## v0.1.0 (2026-04-13)

Initial release of DataLens AI.

### Features

- Natural language to SQL engine with MockProvider and GeminiProvider
- DuckDB-powered in-process analytics
- 15+ auto-generated Plotly visualizations
- Smart data profiling with quality scoring
- 4 built-in sample datasets
- Data stories with 5 templates
- Multi-format report export (HTML, PDF, Markdown, JSON, CSV)
- Streamlit web application with 5 tabs
- Click CLI (demo, explore, query, list-providers, list-datasets)
- Docker support with HuggingFace Spaces deployment
- MkDocs Material documentation
- CI/CD with GitHub Actions
