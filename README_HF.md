# DataLens AI

AI-powered data analytics agent. Upload your data, ask questions in plain English, and get SQL queries, auto-generated charts, and statistical insights.

## Features
- Natural language to SQL (no API keys needed)
- 15+ auto-generated Plotly visualizations
- 4 built-in sample datasets
- Data profiling with quality scoring
- Export reports as HTML, PDF, Markdown, CSV
