# Data Stories

## What Are Data Stories?

Data Stories are auto-generated narrative reports that combine multiple queries, charts, and insights into a cohesive document.

## Story Templates

| Template | Description |
|----------|-------------|
| **Dataset Overview** | Row counts, distributions, top categories, key stats |
| **Trend Analysis** | How metrics change over time |
| **Category Comparison** | Compare performance across groups |
| **Distribution Analysis** | Understand spread and shape of metrics |
| **Correlation Explorer** | Discover relationships between variables |

## Building a Story

1. Go to the **Data Stories** tab
2. Select a template or create a custom story
3. The system executes each question and assembles the narrative
4. Download as HTML, PDF, Markdown, or JSON

## Custom Stories

Provide your own list of questions and a title. Each question becomes a section with:

- SQL query
- Data table
- Auto-generated chart
- Statistical insights

## Export Formats

- **HTML** — styled report with dark theme
- **PDF** — printable report (requires xhtml2pdf)
- **Markdown** — for documentation or wikis
- **JSON** — machine-readable format
- **CSV** — raw data export
