# Changelog

## v0.1.0 (2026-04-13)

Initial release.

### Features

- Natural language to SQL with MockProvider (zero API keys) and GeminiProvider (free tier)
- DuckDB-powered in-process analytics engine
- 15+ auto-generated Plotly visualizations
- Smart data profiling with quality scoring
- 4 built-in sample datasets (E-Commerce, Climate, Stocks, HR)
- Data stories with 5 templates and multi-format export
- Streamlit app with 5 tabs (Data Explorer, Ask Questions, AI Insights, Data Stories, Schema Explorer)
- Click CLI with `demo`, `explore`, `query`, `list-providers`, `list-datasets` commands
- HTML, PDF, Markdown, JSON, CSV report export
- Docker support with HuggingFace Spaces deployment
- MkDocs Material documentation site
- CI/CD with GitHub Actions (lint, test, docs, PyPI publish, HF deploy)
